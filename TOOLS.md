## 常用命令

### Flutter
```bash
# 创建项目
flutter create app_name

# 运行
flutter run

# 构建
flutter build apk
flutter build ios

# 升级依赖
flutter pub upgrade
```

### Git
```bash
# 克隆到工作区
cd ~/.openclaw/workspace-coder/repos
git clone <url>

# 提交
git add .
git commit -m "message"
git push
```

### 代码质量
```bash
# 格式化
dart format .

# 分析
flutter analyze
```

## 浏览器自动化

- **PinchTab**：浏览器控制

## TTS 语音

- **系统自带**：Tingting（女声）、Eddy（男声）
- **命令**：`say -v "Tingting" -r 180 "你好"`
