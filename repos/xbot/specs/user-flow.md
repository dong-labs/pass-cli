# XBot 用户流程设计（Telegram 风格）

> 版本：2.0
> 更新时间：2026-03-17
> 设计理念：对标 Telegram Bot 体验

---

## 1. 设计理念

### 1.1 Telegram Bot 核心机制

| 机制 | Telegram | XBot |
|------|----------|------|
| **创建 Bot** | @BotFather + 命令 | App 内命令 + 可视化 |
| **Token** | 立即生成 | 立即生成 |
| **审核** | 无 | 无 |
| **管理** | 命令交互 | 命令 + 卡片 |
| **接入** | API/Webhook | 小程序码 |

### 1.2 XBot 核心原则

1. **极简** - 最少点击，命令优先
2. **即时** - 创建即用，无需等待
3. **Token 中心** - Token 是唯一凭证
4. **双模式** - 命令模式 + 可视化模式

## 5. Token 机制

### 5.1 Token 格式

```
xbot_<32位随机字符串>

示例：
xbot_abc123def456ghi789jkl012mno345pqr
xbot_XyZ987wVu654tSr321qPo876nMl543kJ
```

### 5.2 Token 用途

| 用途 | 说明 |
|------|------|
| **小程序绑定** | 微信扫码时识别 Bot |
| **API 调用** | 开发者调用 XBot API |
| **Webhook 配置** | 开发者配置回调地址 |

### 5.3 Token 安全

```
• Token 只显示一次，用户需要保存
• 可以用 /revoke 重新生成
• 重新生成后旧 Token 立即失效
• 渠道绑定自动更新
```

## 9. 异常处理

### 9.1 创建失败

```
用户：/newbot

系统：
🤖 好的，创建一个新 Bot！

第一步：给 Bot 起个名字

用户：ab（太短）

系统：
❌ 名称至少需要 3 个字符

请重新输入名称：
```

### 9.2 Token 失效

```
用户在小程序使用

小程序：
❌ Bot Token 已失效

可能原因：
- Token 被重置
- Bot 被删除

请联系 Bot 所有者获取新 Token
```
