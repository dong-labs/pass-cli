# XBot App

> XBot Flutter App - OpenClaw Bot 客户端

---

## 快速开始

### 1. 安装依赖

```bash
flutter pub get
```

### 2. 运行

```bash
flutter run
```

---

## 功能

- ✅ Bot 列表
- ✅ 创建 Bot
- ✅ 对话界面
- ✅ 实时消息

---

## 配置

修改 `lib/config.dart` 设置后端地址：

```dart
static const String wsUrl = 'ws://your-server:3001';
```

---

## 构建

```bash
# Android
flutter build apk

# iOS
flutter build ios
```
