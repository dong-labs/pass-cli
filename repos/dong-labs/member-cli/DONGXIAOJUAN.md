# 咚小圈 - 添加会员指南

> 给咚咚力圈智能体的标准化添加会员流程

---

## 快速开始

```bash
dong-member add <姓名> --wechat <微信号> --price <价格> [可选参数]
```

---

## 必填参数

| 参数 | 说明 | 示例 |
|------|------|------|
| `name` | 会员姓名（第一个参数） | `郭铭` |
| `--wechat, -w` | 微信号 | `guoming9335` |
| `--price` | 金额（默认 499） | `199` |

---

## 可选参数

### 基本信息

| 参数 | 说明 | 示例 |
|------|------|------|
| `--phone` | 手机号 | `13800138000` |
| `--email, -e` | 邮箱 | `user@example.com` |
| `--account, -a` | 账号 ID | `user_001` |
| `--type, -t` | 会员类型 | `yearly`（默认）/ `lifetime` |
| `--project` | 项目 | `donglijuan`（默认） |
| `--join, -j` | 加入日期 | `2026-03-30`（默认今天） |
| `--expire, -x` | 到期日期 | `2027-03-30`（年费默认一年后） |
| `--source, -s` | 来源 | `微信用户` |

### 会员画像（新增）

| 参数 | 说明 | 示例 |
|------|------|------|
| `--region, -r` | 地区 | `山西运城` |
| `--job` | 职业 | `乡村老师` |
| `--tech-level` | 技术水平 | `beginner` / `intermediate` / `advanced` |

### 需求收集（新增）

| 参数 | 说明 | 示例 |
|------|------|------|
| `--needs` | 核心需求（逗号分隔） | `课件制作,公文写作` |
| `--comm` | 本次沟通摘要 | `老用户，想提升工作效率` |

---

## 使用示例

### 示例 1：快速添加

```bash
dong-member add 白禾二少 --wechat augustiliar --price 499
```

### 示例 2：带会员画像

```bash
dong-member add 郭铭 --wechat guoming9335 --price 199 \
  --region "山西运城" \
  --job "乡村老师" \
  --tech-level intermediate
```

### 示例 3：完整信息

```bash
dong-member add 郭铭 --wechat guoming9335 --price 199 \
  --region "山西运城" \
  --job "乡村老师" \
  --tech-level intermediate \
  --needs "课件和思维导图,公文写作,数据安全" \
  --comm "老用户，资深铁粉，想提升备课效率"
```

## 咚小圈工作流程

### 1. 识别意图

用户说："有个人叫郭铭，微信号 guoming9335，山西运城的乡村老师..."

### 2. 提取信息

```json
{
  "name": "郭铭",
  "wechat": "guoming9335",
  "region": "山西运城",
  "job": "乡村老师",
  "price": 199,  // 从对话推断或询问
  "needs": ["课件制作", "公文写作"],  // 从对话提取
  "communication": "老用户，资深铁粉"  // 总结对话
}
```

### 3. 调用命令

```bash
dong-member add 郭铭 --wechat guoming9335 --price 199 \
  --region "山西运城" --job "乡村老师" \
  --needs "课件制作,公文写作" \
  --comm "老用户，资深铁粉"
```

### 4. 确认结果

```
✅ 会员添加成功！
ID: 8
姓名: 郭铭
微信: guoming9335
到期: 2027-03-30
```

## 查看会员

```bash
# 查看所有会员
dong-member list

# 搜索会员
dong-member search 郭铭

# 查看统计
dong-member stats
```

---

## 注意事项

1. **微信号为必填项**：用于唯一标识会员
2. **价格灵活**：可以根据情况调整（如老用户优惠）
3. **需求提取**：从对话中智能提取核心需求
4. **沟通记录**：每次互动都要记录摘要
5. **JSON 输出**：所有命令默认返回 JSON，方便解析
