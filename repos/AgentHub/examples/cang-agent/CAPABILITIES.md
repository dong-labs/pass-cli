# 能力说明

## 核心能力

### 1. 记账

通过 `cang-cli` 记录收支：

```bash
cang record --amount 50 --category 餐饮 --note 午餐
```

**支持输入格式：**
- 自然语言："花了50块吃饭"
- 结构化：`amount + category + note`
- 语音/图片：待实现

### 2. 查询

查询历史记录：

```bash
cang list --period week
cang list --category 餐餐 --month 2026-03
```

### 3. 分析

生成消费分析：

```bash
cang analyze --month 2026-03
cang trend --period 3m
```

### 4. 预算提醒

每日检查预算状态，超支时主动提醒。

| cang-cli | >= 0.1.0 | pip install cang-cli | 记账后端 |

## 文件访问权限

| 操作 | 路径 | 用途 |
|------|------|------|
| 读取 | ~/Documents/finance/** | 导入历史数据 |
| 读取 | ~/.cang/data/** | 读取记账数据 |
| 写入 | ~/.cang/reports/** | 生成分析报表 |

## 能力边界

### 能做
- 记录和查询收支
- 分析消费趋势
- 预算提醒
- 生成报表

### 不能做
- 银行账户直接对接（安全考虑）
- 投资建议
- 税务筹划
- 跨币种自动换算（需要外部 API）
