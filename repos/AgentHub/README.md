# Agent Hub

> 让每个人都能拥有真正懂自己的智能体

## 愿景

**完整智能体 = 人格 + 记忆 + 能力 + 数据**

不是冷冰冰的工具，而是有性格、有记忆、能操作你数据的智能伙伴。

## 核心理念

- **Agent First** - 智能体优先
- **完整人格** - 有灵魂的 AI
- **数据主权** - 本地私有
- **开放生态** - 通用规范
